Node + Expressjs + mysql2

# Introduce

This project is based on Node + ExpressJs + Mysql2.

A Node Server

- Node 12.11.1

# Install

```
npm install
```

# Usage

```
npm run start
```

Then you can visited the Demo page on Browser

```
http://localhost:3001/
```

or

visited the Interface on Postman

```
post: http://localhost:3001/userName
```
